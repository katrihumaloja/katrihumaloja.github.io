Projektit

Kirjaseurantasovellus

React Native Expo Firebase

Tässä projektissa teimme mobiilisovelluksen, joka toimii kirjaseurantasovelluksena. Sovelluksen avulla käyttäjä voi lisätä kirjoja listoille kuten "Luettu", "Luen nyt" ja "Luettavana". Kirjat haettiin ulkoisesta API:sta. Sovelluksessa on myös Lukutilastot-ominaisuus, joka näyttää käyttäjälle tietoja, kuten luettujen kirjojen ja sivujen määrän vuodessa. Tietokanta ja käyttäjätiedot tallennettiin Firebaseen, ja sovellus hyödyntää Firebasea autentikoinnissa. Projektissa käytettiin React Nativea ja Expoa mobiilisovelluksen kehittämiseen. Tämä projekti auttoi minua syventämään React Native ja Firebase -osaamistani.

React MySQL Node.js

Verkkokauppa

WordPress Azure

Tässä projektissa työskentelimme toimeksiantajan kanssa, joka valmistaa ja myy koruja. Tehtävämme oli kehittää verkkokauppa WordPressin avulla, joka olisi helppokäyttöinen ja esteettinen alusta korujen myyntiin. Verkkokauppaa ei kuitenkaan koskaan julkaistu toimeksiantajan kiireiden vuoksi, mutta projekti tarjosi meille arvokasta kokemusta verkkokauppojen kehittämisestä WordPressillä.

Yatzy

React Native Expo

Tässä projektissa rakensin mobiilisovelluksen, joka simuloi Yatzy-pelin pelikokemusta. Sovelluksessa on toteutettu muun muassa noppien heittäminen ja pistelaskenta. Pelaajat voivat tallentaa pisteensä ja tarkastella tilastoja. Projektissa hyödynnettiin React Nativea ja Expoa mobiilisovelluksen kehittämiseen.
